jQuery Spam Detector plugin(spam validator) -Beta Version

This plugin is in BETA stage , package will be refactored and improved in future releases.
Its a basic Manual Spam detection Plugin that detects presence of Spam contents, based on
some predefined(most can be set by users) rules and norms . This plugin has got plenty of
settings according to various needs of users.

Settings / Configurations
spamWordsInUri
: Spam words in URI that needs to restricted.
bannedWords
: Vulgar or Profane words that needs to restricted.
alertText
: The warning text.
onlyReturnStatus
: Default (false),If enabled it will disable all other warning display, and will only return (TRUE for spam presence, False for no spams). Useful when want to include this with other validator options
spamLevel
: The max level on which content can be considered as spam.(recommended 2).
textLimit
: Text limit ,(disable : False).
checkOnType
: default(true) on keypress even shows the message if spam
displayWarning
: default(true), Turn off warning message or not
maxUrlAllowed
: The number of URLs allowed in content
maxUrlLength
: Url length
formObj
: Can be given 'ID' or 'Class' or 'form' if to check on submit form validation , else leave empty to disable .
errorTextContainer
: Error div Id( can be class also).
limitTextContainer
: Limit div Id can be class also).
warningStyle
: Style of warning Text.
limitStyle
: Style of Limit Text.

(**Must include <script src="spamDetector.js type="text/javascript" ></script>)
